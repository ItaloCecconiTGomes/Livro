# 📚 Livro Virtual

> Projeto desenvolvido para praticar Programação Orientada a Objetos (POO) em Java.

> Simula o manuseio de um livro virtual.

## 💻 Tecnologias utilizadas:
* Java
* Eclipse IDE
* Git e GitHub

## ⚙ Funcionalidades:
* 🤵 Criar pessoa
* 📘 Selecionar o livro
* 📖 Abrir
* 📕 Fechar
* 🔖 Folhear
* ⏩ Avançar página
* ⏪ Voltar página
